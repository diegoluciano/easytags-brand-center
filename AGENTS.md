# EasyTags Brand Center — Agent Instructions

## Project goal

Build and maintain a static Portuguese Brand Resource Center for EasyTags, a company that sells electronic shelf labels for small and medium-sized supermarkets.

## Brand direction

The design should feel:
- modern
- technological
- trustworthy
- clean
- retail-oriented
- suitable for supermarket owners and future UI/UX/product teams

## Brand colors

- Primary green: #74C63E
- Dark charcoal: #2E2E2E
- White: #FFFFFF

Use green as an accent/action color, not as an overwhelming background color.

## Typography

Use:
- Inter for general interface and text
- Oswald for condensed retail-style headings, prices, digital tags, and campaign headlines

## Code rules

- Keep the project simple and static.
- Use HTML, CSS and vanilla JS.
- Do not add a framework unless explicitly requested.
- Keep CSS in css/styles.css.
- Keep JS in js/main.js.
- Use semantic HTML.
- Write code comments in English.
- Do not remove working sections unless specifically requested.
- Prefer clean, maintainable code over clever abstractions.

## Content language

All visible website content should be in Portuguese.

## Expected sections

- Visão geral
- Logos
- Paleta de cores
- Tokens de sistema
- Tipografia
- Manual de aplicação
- Loading animation
- Mockups
- Criativos base para Instagram
- Downloads

## Deployment

The project should be ready for GitHub Pages.
